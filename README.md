# Vibration Sensor With LCD And Buzzer
Vibration Sensor With I2C LCD And Buzzer.

## Prerequisites
1. Arduino Uno R3
2. Arduino IDE
3. Vibration Sensor Module
4. Breadboard / PCB
5. Buzzer
6. LCD Screen With I2C
7. Jumper Wires
